### Installation

1. Clone the repo
```sh
git clone https://github.com/eliottoblinger/auth-secu.git
```
2. Go to folder
  ```sh
  cd auth-secu
  ```
3. For back
  ```sh
  cd back
  npm install
  cp .env.example .env
  ```
4. For front
  ```sh
  cd front
  npm install
  ```

### Run app

1. Run back
  ```sh
  cd back
  node index.js
  ```
2. Run front
  ```sh
  cd front
  npm run dev
  ```
